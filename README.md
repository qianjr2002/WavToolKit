# WavToolKit
音频处理小工具

## 文件列表

- [wav_info_show.py](./wav_info_show.py)
    - 详细显示wav文件的信息

- [resample_to_16kHz.py](./resample_to_16kHz.py) 
    - 将wav文件重新采样到16kHz的脚本

- [plot_wav.py](./plot_wav.py)
    - 画出单通道wav文件的时域波形图以及频域功率谱图

- [calculate_wavfile_SI-SNR.py](./calculate_wavfile_SI-SNR.py)
    - 计算一对wav文件的SI-SNR值
